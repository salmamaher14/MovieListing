A simple iOS app built with Swift that displays a list of top-rated movies using a clean and user-friendly interface.  
The app fetches movie data from a remote API and showcases movie posters, details, and more.

---

## 📱 Features

- 📃 List of top movies  
- 🖼️ Asynchronous loading of movie posters  
- 🔍 Movie details screen  
- 🎨 Uses Kingfisher for efficient image caching  

---

## 🛠️ Technologies Stack

- Swift  
- UIKit  
- Kingfisher for image loading  
- Combine for async network calls  
- MVVM architecture design pattern

## 🔧 Setup Instructions

1. Clone the project from GitHub.
2. Open the `.xcodeproj` in Xcode.
3. It’s tested on **iPhone 15 Pro simulator**, but feel free to use any screen size.
4. Build and run the app.
